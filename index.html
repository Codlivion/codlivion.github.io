My name is Ahmet. I'm a self-learning programmer and indie game developer living in Turkey.


I am occupied with developing and publishing my own games, which include some 8bit-16bit style platformers and RPGs.

I've been actively working on these projects since 2022, learning programming and improving my skills at game design, drawing, pixel art, graphic design and sound design.


I work primarily with C# and C++, especially in game development, with frameworks like MonoGame and PixelGameEngine.

I've been learning and practicing other fields of programming too. I have been experimenting with Windows Forms, ASP.NET, Neural Networks and HTML/CSS/JavaScript.


The site consists of three sections:


Developer Logs: where I will be sharing articles and guides on game development and programming topics.


CoDream: where I will be sharing news on my main game projects.


Browser Games: where you can play some simple games on the browser, written in JavaScript or built for WASM
